# Hall of Fame

This page is reserved for security researchers and contributors who responsibly disclose vulnerabilities to Invoice Liquidity Network (ILN).

Entries should be added by maintainers after a fix is merged and the disclosure has been publicly announced.

## How to add an entry

- Add the researcher handle or name
- Add the date of disclosure or fix publication
- Add the severity of the finding
- Add a brief description of the issue after the fix is deployed
- Add a CVE or advisory link if applicable

### Entry format

`Name / Handle — Date — Severity — Brief description — CVE/Advisory link`
