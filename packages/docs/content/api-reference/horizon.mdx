# Horizon API Reference

While the ILN SDK abstracts most interactions, you may need to query Stellar's Horizon API directly for historical data and analytics.

## Base URLs

| Network | URL |
|---------|-----|
| Testnet | `https://horizon-testnet.stellar.org` |
| Mainnet | `https://horizon.stellar.org` |

curl "https://horizon-testnet.stellar.org/ledgers/{ledger_sequence}/effects?type=contract_event"

Transaction History
Get all transactions involving the ILN contract:
curl "https://horizon-testnet.stellar.org/accounts/{CONTRACT_ID}/transactions?order=desc"

Account Balances
Check token balances for any address:
curl "https://horizon-testnet.stellar.org/accounts/{ADDRESS}"

Response includes all trustlines and balances:
{
  "balances": [
    {
      "balance": "1000.0000000",

Rate Limits
| Endpoint    | Limit               |
| ----------- | ------------------- |
| Horizon     | 3600 requests/hour  |
| Soroban RPC | 100 requests/minute |


Pagination
All list endpoints use cursor-based pagination:
# Get next page
curl "https://horizon-testnet.stellar.org/accounts/{ID}/payments?cursor={cursor}&limit=200"

# Get previous page
curl "https://horizon-testnet.stellar.org/accounts/{ID}/payments?cursor={cursor}&order=asc&limit=200"
