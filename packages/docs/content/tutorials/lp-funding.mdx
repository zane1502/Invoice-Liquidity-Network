# LP Funding Guide

Learn how to provide liquidity to ILN invoices and earn yield.

## What You'll Learn

- How to evaluate invoice opportunities
- How to fund invoices via CLI and SDK
- How to track your LP positions
- Risk management strategies

## Prerequisites

- LP wallet with USDC on Stellar testnet
- Understanding of invoice discount rates

## Understanding Returns

When you fund an invoice, you earn the discount rate as yield:

Investment: 97 USDC
Return: 100 USDC
Profit: 3 USDC (3.09% return)
Annualized: ~37% APY (for 30-day invoice)

Risk Management

Diversification
Don't put all funds in one invoice. Spread across:
- Multiple issuers
- Different due dates
- Various discount rates

Due Diligence
- Check issuer reputation on-chain
- Verify payer identity when possible
- Start with smaller amounts

Exit Strategy
- Invoices are non-transferable until secondary market launches
- Only fund amounts you can hold until due date
- Consider grace period (7 days) in your planning


Automated Funding (Advanced)
Set up auto-funding rules:
import { ILNClient } from '@invoice-liquidity/sdk';

const client = new ILNClient({ /* config */ });

// Auto-fund invoices matching criteria
client.onInvoice('submitted', async (invoice) => {
  if (
    invoice.discountRate >= 300 &&    // At least 3% discount
    invoice.amount <= 1000_0000000 &&  // Max 1000 USDC
    invoice.dueDate - Date.now() < 86400 * 60 // Due within 60 days
  ) {
    await client.fundInvoice(invoice.id);
    console.log(`Auto-funded invoice ${invoice.id}`);
  }
});
